# Déjame buscar eso por ti

Una herramienta divertida para guiar búsquedas, pensada para solucionar el problema de quienes tienen poca maña con el navegador . Soporta los tres principales motores de búsqueda: Baidu, Google y Bing.

## Características

* Compatibilidad con los tres principales motores de búsqueda:

  * 百度 (Baidu)
  * 谷歌 (Google)
  * 必应 (Bing)
* Interfaz de usuario limpia e intuitiva
* Genera enlaces de búsqueda compartibles
* Permite copiar el enlace de búsqueda con un solo clic
* Incluye función de vista previa de búsqueda

## Cómo usar

1. Elige el motor de búsqueda que quieras usar (Baidu / Google / Bing).
2. Escribe tu consulta en el cuadro de búsqueda.
3. Haz clic en el botón de búsqueda.
4. Obtén el enlace generado; puedes:

   * Copiar el enlace para compartirlo con otros.
   * Hacer clic en "Vista previa" para ver cómo se muestra la búsqueda.

## Stack tecnológico

* HTML5
* CSS3
* jQuery
* Clipboard.js
